# Walmart-Sales-Data-Analysis
Project by: Hafsa Khalil
# Project Overview
The goal of this project was to analyze Walmart’s retail data to identify key business drivers, specifically focusing on the impact of holiday events on weekly sales performance.
# Key Findings
Holiday Impact: Sales during holiday weeks are, on average, 7.84% higher than regular weeks.
Business Insight: This suggests that Walmart needs to increase inventory levels and staffing by at least 8-10% during these periods to meet the increased demand.
# Tools Used
Google Sheets / Excel: Pivot Tables, IF Statements, and Data Visualization.
Data Cleaning: Handled missing data and formatted date/currency fields for analysis.
