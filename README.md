Walmart-Sales-Data-Analysis

Project by: Hafsa Khalil

Project Overview

The goal of this project was to analyze Walmart’s retail data to identify key business drivers, specifically focusing on the impact of holiday events on weekly sales performance. The analysis was performed using both Excel and Python to provide complementary insights.

Excel Analysis

Tools Used: Google Sheets / Excel

Analysis Performed:

Pivot Tables for summarizing weekly sales

IF statements for conditional calculations

Charts for visualizing trends during holidays vs regular weeks

Data cleaning: handled missing values, formatted dates and currency fields

Key Findings:

Holiday Impact: Sales during holiday weeks are, on average, 7.84% higher than regular weeks

Business Insight: Walmart should increase inventory levels and staffing by at least 8-10% during holidays to meet demand

Python Analysis

Tools Used: Python 3, Jupyter Notebook, pandas, numpy, matplotlib, seaborn, scikit-learn

Analysis Performed:

Data cleaning and preprocessing

Exploratory Data Analysis (EDA) to find trends and patterns

Visualization of correlations between Weekly Sales, Temperature, Fuel Price, CPI, and Unemployment

Linear regression model to forecast weekly sales

Files:

Walmart_Data_Analysis.ipynb — Python notebook with full code and visualizations

Insights:

Confirmed that holidays boost weekly sales

Unemployment negatively correlates with sales

Temperature has minimal effect on sales

Linear regression model provides a baseline for sales forecasting

Dataset

Walmart Data Analysis and Forecasting.xlsx — Raw and cleaned dataset used for both Excel and Python analysis
